# MSE 245 Lecture Exercises

Code exercises based on the provided MSE 245 lecture deck, organized one file per lecture.

Notes:
- These exercises are derived from the lecture topics visible in the uploaded PDF.
- Each file contains scaffolded tasks and TODOs for students to complete.
- Files are named by lecture order and topic.

## Included lectures
- Lecture 1: The Web: A Brief History
- Lecture 2: JavaScript Foundations II
- Lecture 3: Asynchronous Programming
- Lecture 4: React JSX and Components
- Lecture 5: React State and Effects
- Lecture 6-1: UI Components, npm, and App Structure

## Suggested usage
- Open a file in `exercises/`
- Read the top comment
- Complete each TODO in order
- Run and test your solution locally
